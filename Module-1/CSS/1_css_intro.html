What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS gives style to raw HTML
  • CSS describes how HTML elements are to be displayed on screen
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • CSS makes web page responsive
  • HTML gives the structure while CSS gives the design
  • CSS makes our website beautiful and morden looking

  • CSS is categorised into three types
    • Inline: added to the element directly using style attribute.
    • Internal: kept in side the html page under head tag, using <style> open </style> close.
    • External: css is kept seperately ina .css file and linked to the html pages.
